Open CSD

'OPEN CSD'는 기술공유 홈페이지입니다.

페이지는 총 8페이지로 이루어져 있으며 pc와 mobile 각각 제작되었습니다.

그누보드 기반의 사이트로 게시판 연결작업을 하였으며, 서브페이지는 고객의 니즈에 맞게 퍼블리싱하였습니다.
메인 페이지에서 OPEN CSD의 기술에 대해 아이콘과 텍스트로 나열하여 볼 수 있게 하였으며,
'공지사항', '공개SW' 게시판의 최신 등록된 글을 가져와 바로 볼 수 있게 하였습니다.
jQuery 플러그인 사용을 통해 협력사 로고를 자연스럽게 흐르게 하여 볼 수 있게 하였으며,
로고 영역으로 마우스오버 되면 scroll멈추고, 마우스아웃이면 다시 scroll됩니다.
협력사 로고 클릭 시 해당 사이트로 이동하며, 새로운 탭에서 사이트 확인이 가능합니다.
